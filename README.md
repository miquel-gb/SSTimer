# SSTimer
## Releases
Releases can be downloaded from inside Releases folder located in project's root.

## Project structure
The project uses a simple structure, as it's just one page, all the computing is done in a single JS file.
```
root
│   config.xml
│   package.json  
│
└─── www
│   │   index.html
│   │
│   └─── css
│   |      index.css
│   |   
│   └─── img
│   |      logo.png
│   |      flash.jpg
│   |   
│   └─── js
|          index.js
|   
└─── releases
        SSTimer_1.0.0-apk
```
